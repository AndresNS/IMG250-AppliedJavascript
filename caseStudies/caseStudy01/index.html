For case study #1 you will adopt a process similar to what has been demonstrated in class and through lab to determine the underlying causes of a particular JavaScript oddity, as evidenced by the code snippet below:

var message = 'foo';

function printIt() {
  console.log(message);
  var message = 'bar';
};

printIt();

Please enter the code snippet above in to your browser console, or run it through an external .js file to see the result. In examining the code, you might expect that the result of running the printIt function would be printing either 'foo' or 'bar' to the console; however, this is not the case. It is your responsibility to determine why and to identify reputable publications or commentary online to help you explain it.

To complete this assignment you must:

  • Find at least three reputable sources online to help you explain the code snippet above (more sources are certainly acceptable).
  • Clearly explain the code snippet above and the result in your own words and with reference to your three sources. I expect this to be clearly written and include proper references. Your write up should be no less than 200 words, but may be up to 400.
  • Develop a related, but distinct example demonstrating the same result, but using different code. Make sure to explain how your example is similar and how it is different from the code snippet above.
